    Diseño web

    ¿Qué haré como diseñador web?

    Me encargaré de aplicar y combinar de manera dinámica los diferentes elementos que forman parte de un sitio web: como colores, tipografías, menú, botones, etc. Así mismo, estaré al tanto de las tendencias actuales en diseño web para aplicarlas correctamente en las creaciones, adaptándolas a cada proyecto, cliente y situación.

    ¿Que ofrezco?

    • Diseño, funcionalidad y maquetación de sitios web.
    • Pregrado de diseño web o afines.
    • Diseño UX/UI.
    • Diseño web responsive.

  • Desarrollo web profesional

    Desarrollo web front-end

    ¿Qué haré como desarrollador front end?

    Trabajaré de “lado del cliente”. Me ocuparé del diseño web y de las funciones con las que los usuarios interactúan en forma directa. En pocas palabras, procuraré interacciones eficientes, una buena experiencia de navegación y que los sitios sean visualmente atractivos.

    ¿Que ofrezco?

    • Me encargaré del diseño y desarrollo de sitios web.
    • Diseños funcionales, usables y placenteros
    • Pruebas de usabilidad
    • Estrategias

  • Mantenimiento de paginas web

    Mantenedor web.

    ¿Que hace un mantenedor web?

    Básicamente se encarga de tener al día la pagina web, mantener actualizaciones al día, evitar fallos y estar en constante revision de la misma, todo esto llevarlo adecuadamente para ahorrar tiempo y trabajar con seguridad.

    ¿Que ofrezco?

    • Garantizar la seguridad del sitio web, eliminar vulnerabilidades.
    • Realización de ajustes en el servidor para optimizar tiempos de carga.
    • Programación de formularios, suscripciones...
    • Tener la página web actualizada y mejorada.
